# 線框圖：後台案件分派 (Admin Dispatch)

**Route**: `/admin/dispatch`
**Goal**: 管理員透過地圖與列表，快速將新進案件分派給承辦人。

```text
+---------------------------------------------------------------+
| [Filter Bar]                                                  |
| 狀態: [新進案件 x] [未分派 x]  區域: [全部 ▾]  日期: [今日 ▾]   |
+---------------------------------------------------------------+
| [Split View Container]                                        |
|                                                               |
| [Left: Case List (30%)]      | [Right: Map View (70%)]        |
|                              |                                |
| 1. [一般/A123001] [新進]      |                                |
|    板橋區中山路...            |         🔴(A123001)            |
|    <由民眾通報>               |                                |
|    [詳細] [分派]              |             🔴(A123002)        |
| ---------------------------- |                                |
| 2. [蜂案/B999001] [緊急]      |      🏫 市政府                 |
|    土城區金城路...            |                                |
|    <由119轉介>                |           🟢(Caseworker A)     |
|    [詳細] [分派]              |                                |
|                              |                                |
| ---------------------------- | [Map Controls]                 |
| [Pagination < 1 2 3 >]       | [Show Heatmap] [Show Workers]  |
|                              |                                |
+------------------------------+--------------------------------+
```

**分派 Modal (點擊 [分派] 後彈出)**:
```text
+--------------------------------------------------+
| 分派案件：A123001                                  |
+--------------------------------------------------+
| 案件地點：板橋區中山路一段...                        |
| 建議承辦人 (基於距離/負荷)：                          |
|  [x] 王小明 (距離 1.2km, 手上 2 案)                 |
|  [ ] 李大華 (距離 3.5km, 手上 0 案)                 |
|                                                  |
| 指派說明：                                         |
| [_______________________]                        |
|                                                  |
| [ 取消 ]                    [ 確認分派 ]           |
+--------------------------------------------------+
```
