# 線框圖：前台案件地圖 (Public Map View)

**Route**: `/map`
**Goal**: 讓民眾查看周邊案件分布，了解動保處處理現況 (公開透明)。

```text
+---------------------------------------------------------------+
| [Navbar] (引用 00_Global_Layout)                              |
| [Logo]   [首頁] [案件通報] [進度查詢] [案件地圖] [教育資源]   |
+---------------------------------------------------------------+
| [Floating Search Bar]                                         |
|  [ 輸入路名或區域... 🔍 ]                                      |
+---------------------------------------------------------------+
|                                                               |
|  [ Full Screen OSM Map ]                                      |
|                                                               |
|       📍 (案 A-處理中)                                         |
|                                                               |
|                       📍 (案 B-已結案)                         |
|                                                               |
|                                                               |
|            (User Location 🔵)                                 |
|                                                               |
|                                                               |
+---------------------------------------------------------------+
| [Legend / Filter (Bottom Sheet on Mobile)]                    |
| [x] 處理中 (紅)  [x] 已結案 (綠)                              |
| 顯示最近 7 天案件                                              |
+---------------------------------------------------------------+
```

**隱私保護**:
*   公開地圖上的點位 **不顯示詳細地址** (僅顯示路段，如 "中山路一段附近")。
*   不顯示報案人資訊。
*   不顯示敏感案件 (如虐待動物調查中案件)。
