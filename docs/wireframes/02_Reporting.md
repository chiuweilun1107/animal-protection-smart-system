# 線框圖：案件通報頁 (Reporting Page)

**Route**: `/report/general` 或 `/report/bee`
**Goal**: 引導民眾完成通報，收集必要資訊 (含 GIS & 照片)。

```text
+---------------------------------------------------------------+
| [Navbar] < 返回                                               |
+---------------------------------------------------------------+
| [Step Process Bar]                                            |
| (1) 基本資料 -> (2) 案件詳情 -> (3) 確認送出                   |
+---------------------------------------------------------------+
|                                                               |
| [Section: 地點定位]                                            |
|  說明：請拖動地圖釘選案發位置。                                  |
|  +---------------------------------------------------------+  |
|  | [ Map View (OSM) ]                                      |  |
|  |    📍 (Draggable Pin)                                   |  |
|  |                                                         |  |
|  | [O] 定位我的位置                                         |  |
|  +---------------------------------------------------------+  |
|  地址概估：[ 新北市板橋區中山路一段161號 _________ ] (自動帶入)   |
|                                                               |
| [Section: 案件資訊]                                            |
|  案件類型： [ 一般案件 ▾ ] (唯讀，依入口而定)                    |
|  *現場照片：                                                   |
|  [ + 上傳照片 ] (支援多張預覽, Drag & Drop)                     |
|                                                               |
|  *詳細描述：                                                   |
|  [__________________________________________________]         |
|  [__________________________________________________]         |
|  (Placeholder: 請描述動物特徵、蜂巢大小、位置...)                 |
|                                                               |
| [Section: 聯絡人 (非必填，但建議)]                              |
|  姓名：[_________]  電話：[_________]                          |
|                                                               |
| [Section: 驗證]                                                |
|  *驗證碼： [ 1234 ] (Image)  輸入：[____]                       |
|                                                               |
| [Action Bar]                                                  |
| [ 取消 ]                         [ 下一步 / 送出通報 ]          |
|                                                               |
+---------------------------------------------------------------+
```

**蜂案特殊欄位 (若為 Bee Mode)**:
*   增加「蜂巢大小」選擇：(拳頭大 / 籃球大 / 輪胎大)
*   增加「蜂巢位置」選擇：(樹上 / 屋簷 / 地面 / 其它)
